# React.JS Blog Application

The React.js Blog Application is a web project that allows users to list and view blog posts in detail with a modern user interface.
The application is built on powerful technologies such as React, Bootstrap, Zustand, Axios, and i18next. The goal is to enable users to easily read blog content, share posts, and utilize multilingual support.

3. Run DevOps environment

# Developers Notes

- Global state management is provided in the `src/store` directory using `Zustand`.

- Axios configuration is done in the `src/services/api.js` file.

- Multi-language configuration is located in the `src/i18n.js` file.

- The `src/styles/main.scss` file is used to customize Bootstrap classes.
